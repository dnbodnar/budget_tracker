# Budget Tracker ETL Pipeline

A personal finance tracking system using Python, PySpark, PostgreSQL, and Apache Airflow

## Achitectuire 
- Extract: Parse credit card transaction emails 
- Transform: Clean and categorize using PySpark
- Load: Store in PostgreSQL database 
- Orchestrate: Automate with Apache Airflow

## Tech Stack 
- Python 3.13.7
- PySpark
- PostgreSQL
- Docker
- Apache Airflow
